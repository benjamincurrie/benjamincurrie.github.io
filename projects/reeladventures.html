Reel Adventures

Website & Social Media

Reel Adventures is bringing the best in adrenalin packed movies to Perth, no matter what gets your blood pumping we have something for you. We screen the best outdoor films featuring Mountain Biking, Rock Climbing, Kayaking, Mountaineering and more.

Reel Adventures is a personal project of mine and I used it as an experiment to create a Wordpress theme with the fantastic Twitter Bootstrap CSS Framework. A fun project that never really went beyond one screening, it lies as a testimate to what could have been.

Role: Content Writing, Web Design & Implementation
Technologies: Photoshop, Wordpress, HTML, CSS, Javascript
